VIRTUAL PIANO


Use the "tab" -> "\" keys for white notes

And "1" -> "backspace" keys for black notes (except "3", "7", "0")





Source code: https://github.com/sax1476/Virtual-Piano/

Notes sample: TEDAgame (at https://freesound.org/people/TEDAgame/packs/25405/ )

P/s: All of your records will be save to the browser's local storage
